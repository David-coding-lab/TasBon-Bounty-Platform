# TasBon-Bounty-Platform

TasBon is a community-driven bounty platform that connects task creators with skilled developers. Post bounties, complete missions, and earn rewards in a structured, escrow-backed environment.

Current setup: React + Vite + Tailwind CSS.

Team workflow and contribution guidelines are in [Team-Convention.md](Team-Convention.md).
The React app scaffold will create `src/`; we organize the feature-based structure inside that folder after the app is generated.
