IMAGE GENERATION API - POLLINATIONS.AI
Free, no API key required.

ENDPOINT (Image):
https://image.pollinations.ai/prompt/{prompt}

USAGE:
curl "https://image.pollinations.ai/prompt/your%20description%20here" -o output.jpg

NOTES:
- Returns JPEG image directly
- Use URL-encoded prompts (spaces = %20)
- 768x768 default resolution
- No auth/rate limits for basic use

VIDEO GENERATION - POLLINATIONS.AI
Requires API key (free credits from enter.pollinations.ai)

ENDPOINT (Video):
https://gen.pollinations.ai/video/{prompt}

AUTH:
Header: Authorization: Bearer YOUR_API_KEY

USAGE:
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://gen.pollinations.ai/video/your%20prompt" -o output.mp4

MODELS: Seedance, Veo (alpha)
